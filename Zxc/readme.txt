/*
关于canvas 显示像素问题

storkeRect 是以 左上角坐标+宽高来绘制，

左上角坐标：
    坐标值如果是整数，画出的线会发虚，需要+0.5才能正常显示
但是这样的话画出来的矩形宽和高就会大了1，需要在将宽和高都减
去1。
    但是这样的话就没法画宽高都是1的像素点，其余如在(0,0)处
画宽高都是2的小矩形，以以上计算后的参数绘图，可以正常显示

*/
